# Teslo API
1. Clonar proyecto
2. ```npm i``` o ```yarn```
3. cloanr archivo ```.env.template``` y renombrar a ```.env```
4. cambiar variables de entorno
5. levantar base de datos
```
docker compose up -d
```

6. Levantar: ```npm run start:dev``` o ```yarn start:dev```

7. Ejecutar SEED 
```
http://localhost:3000/api/seed
```
